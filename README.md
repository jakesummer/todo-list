# Todo List

Live Site: https://jakesummer.github.io/todo-list/

## Features
* Create todos with a title, description, due date, and priority
    * Can also edit already made todos
* Create different projects to group todos in
    * Can also edit the name of projects
* Saves projects and todos to local storage so they won't reset on page load
